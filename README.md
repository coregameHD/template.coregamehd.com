# Japanese Learning Tools

A collection of tools for learning Japanese, built with [Eleventy](https://www.11ty.dev/).

## Features

- **Japanese Tools**: General Japanese language tools and utilities
- **Katakana Practice**: Practice and learn Katakana characters
- **Yojijukugo**: Learn Japanese four-character idioms
- **Responsive Design**: Works on desktop and mobile devices
- **Modern UI**: Built with Tabler UI and Bootstrap 5

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm (v6 or higher)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/your-repo.git
   cd your-repo
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

### Development

Start the development server:

```bash
npm start
```

This will start the Eleventy development server at `http://localhost:8080`.

### Building for Production

To build the site for production:

```bash
npm run build
```

The built files will be available in the `_site` directory.
